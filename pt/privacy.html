Resumo rápido

TL;DR: A SENSFLOW não coleta seus dados pessoais identificáveis. Suas configurações ficam salvas apenas no seu próprio navegador. O site pode exibir anúncios de terceiros, mas não compartilhamos dados pessoais com anunciantes nem criamos perfis de usuário para fins publicitários.

Esta página existe para ser transparente sobre como o site funciona tecnicamente e quais dados, se algum, passam por nossos sistemas.

O que coletamos

As ferramentas da SENSFLOW rodam diretamente no seu navegador. Isso significa que não processamos seus dados de sensibilidade em servidores próprios — os cálculos acontecem localmente, no seu dispositivo.

Não coletamos:

  • Nome, e-mail ou qualquer dado de identificação
  • Configurações de sensibilidade ou perfis enviados para servidores
  • Histórico de conversões ou uso das ferramentas
  • Endereço IP (além do que qualquer servidor web registra automaticamente em logs temporários)

Dados locais (localStorage)

As funcionalidades de "Minha Sensibilidade Universal" e "Meu ADS Universal" usam a API localStorage do seu navegador para salvar preferências.

Isso significa que:

  • Os dados ficam exclusivamente no seu dispositivo
  • Não são transmitidos para nenhum servidor
  • Você pode apagar a qualquer momento usando o botão "Limpar" no painel ou limpando os dados do site nas configurações do navegador
  • Não são compartilhados entre dispositivos diferentes

As chaves armazenadas são: sensflow_profile (sensibilidade hipfire) e sensflow_ads_profile (configurações ADS).

Serviços de terceiros

O site carrega as seguintes fontes externas:

  • Google Fonts — para carregar as tipografias Rajdhani, Inter e JetBrains Mono. O Google pode registrar que seu IP fez uma requisição para os servidores de fontes. Consulte a política do Google para mais detalhes.

O site pode utilizar redes de anúncios de terceiros (como Google AdSense) para exibir publicidade. Essas redes podem usar cookies próprios para personalizar anúncios com base em visitas anteriores. Você pode gerenciar essas preferências nas configurações do seu navegador ou via opt-out de anúncios.

Não utilizamos Google Analytics ou Meta Pixel para rastreamento de comportamento de usuários.

Cookies

A SENSFLOW não usa cookies. Nenhum cookie é definido pelo site. As preferências são salvas via localStorage, que é tecnicamente diferente de cookies e não é enviado automaticamente em requisições HTTP.

Seus direitos

Como não coletamos dados pessoais identificáveis, não há um perfil seu para acessar, corrigir ou excluir em nossos sistemas.

Para apagar os dados salvos localmente no seu navegador:

  • Use os botões "Limpar" nos painéis de perfil do site
  • Ou acesse as configurações do seu navegador → Privacidade → Dados do site → SENSFLOW → Excluir
