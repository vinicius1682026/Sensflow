Resumen rápido

TL;DR: SENSFLOW no recopila tus datos personales identificables. Tu configuración se guarda solo en tu propio navegador. El sitio puede mostrar anuncios de terceros, pero no compartimos datos personales con anunciantes ni creamos perfiles de usuario con fines publicitarios.

Esta página existe para ser transparente sobre cómo funciona el sitio técnicamente y qué datos, si los hay, pasan por nuestros sistemas.

Qué recopilamos

Las herramientas de SENSFLOW funcionan directamente en tu navegador. Esto significa que no procesamos tus datos de sensibilidad en servidores propios — los cálculos ocurren localmente, en tu dispositivo.

No recopilamos:

  • Nombre, correo o cualquier dato identificativo
  • Configuraciones de sensibilidad o perfiles enviados a servidores
  • Historial de conversiones o uso de las herramientas
  • Dirección IP (más allá de lo que cualquier servidor web registra automáticamente en logs temporales)

Datos locales (localStorage)

Las funciones "Mi Sensibilidad Universal" y "Mi ADS Universal" usan la API localStorage de tu navegador para guardar preferencias.

Esto significa que:

  • Los datos permanecen exclusivamente en tu dispositivo
  • No se transmiten a ningún servidor
  • Puedes borrarlos en cualquier momento con el botón "Limpiar" del panel, o borrando los datos del sitio en la configuración del navegador
  • No se comparten entre dispositivos distintos

Las claves almacenadas son: sensflow_profile (sensibilidad hipfire) y sensflow_ads_profile (configuración ADS).

Servicios de terceros

El sitio carga los siguientes recursos externos:

  • Google Fonts — para cargar las tipografías Rajdhani, Inter y JetBrains Mono. Google puede registrar que tu IP hizo una petición a sus servidores de fuentes. Consulta la política de Google para más detalles.

El sitio puede utilizar redes publicitarias de terceros (como Google AdSense) para mostrar anuncios. Estas redes pueden usar sus propias cookies para personalizar anuncios según visitas anteriores. Puedes gestionar estas preferencias en la configuración de tu navegador o mediante exclusión de anuncios.

No utilizamos Google Analytics ni Meta Pixel para rastrear el comportamiento de los usuarios.

Cookies

SENSFLOW no usa cookies. El sitio no define ninguna cookie. Las preferencias se guardan mediante localStorage, que es técnicamente distinto de las cookies y no se envía automáticamente en las peticiones HTTP.

Tus derechos

Como no recopilamos datos personales identificables, no existe un perfil tuyo que acceder, corregir o eliminar en nuestros sistemas.

Para borrar los datos guardados localmente en tu navegador:

  • Usa los botones "Limpiar" en los paneles de perfil del sitio
  • O ve a la configuración de tu navegador → Privacidad → Datos del sitio → SENSFLOW → Eliminar
